# Starting the new dataset
spotify_clean <- proj_data %>%
  select(
    Title,
    Rank,
    Artists,
    Danceability,
    Energy,
    Loudness,
    Speechiness,
    Acousticness,
    Instrumentalness,
    Valence
  )

# Creating multi-artist variable so we can easily use it later
spotify_clean <- spotify_clean %>%
  mutate(
    multi_artist = ifelse(str_detect(Artists, ","), 1, 0)
  )

# Now flipping the rank variable because 1 should be the best so it's easy to understand
spotify_clean <- spotify_clean %>%
  mutate(
    rank_score = 201 - Rank
  )

# Trying now to create a "How long the song stayed in the Top 200" variable
spotify_clean <- spotify_clean %>%
  group_by(Title) %>%
  mutate(
    days_in_top200 = n(),
    multi_artist = ifelse(str_detect(Artists, ","), 1, 0)
  ) %>%
  ungroup()

spotify_data <- spotify_clean %>%
  group_by(Title, Artists) %>%
  summarise(
    avg_rank_score = mean(rank_score),
    days_in_top200 = n(),
    multi_artist = first(multi_artist),
    Danceability = first(Danceability),
    Energy = first(Energy),
    Loudness = first(Loudness),
    Speechiness = first(Speechiness),
    Acousticness = first(Acousticness),
    Instrumentalness = first(Instrumentalness),
    Valence = first(Valence),
    .groups = "drop"
  )

Conclusion on Influential Points

After analyzing the influential points in our dataset, a small number of observations had both high leverage and high Cook’s distance (approximately 20 out of nearly 8,000 songs, or less than 0.25% of the data). These songs have unusual combinations of audio features that could impact the regression model. However, we found that removing influential observations yielded nearly identical results, indicating that the model’s conclusions are not overly affected by these points. For the Top 200 model, several audio features, including danceability, loudness, speechiness, valence, and multi-artist status, were consistently significant predictors of how long a song remains in the charts. However, the rank model had extremely low explanatory power (R² ≈ 0.003), suggesting that audio features may be poor predictors of a song’s average ranking. This implies that rank is likely influenced by external factors such as marketing, artist popularity, and social trends rather than musical characteristics alone.
